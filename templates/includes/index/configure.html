Hooray! You have set up the hiway theme.

Next step is to create/configure a section that will show up here.

Show content on the home page:

Use the following template to create content/home/_index.md: 

+++
title = "Home"
template = "index.html"
+++

# Content Goes here

It will appear on the home page / index.

And add this line to your config.toml: 

[extra]
index_content = "home"

Display list of recent posts from your blog:

Create content/blog/_index.md using the following template: 

+++
title = "Blog"
sort_by = "date"
+++

Now create a post at content/blog/first-post.md: 

+++
title = "First Post"
date = 2020-11-20
+++

It begins!



Here it is.

Add this line to your config.toml: 

[extra]
index_preview = ["blog"]

You can add multiple sections to index_preview.
